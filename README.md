## 프로젝트 소개
클라우드 네이티브 풀스택 & DevOps 개발자 교육
클라우드 네이티브 환경에서의 풀스택 애플리케이션 개발 및 DevOps 자동화에 관심이 있는 훈련생은 프로그래밍(Java, JavaScript), 데이터베이스(SQL, MySQL, RDBMS 최적화), 웹 서비스 개발(HTML, CSS, React, Spring Boot), API 개발 및 보안(OAuth, JWT), 컨테이너 기반 배포(Docker, Kubernetes), 클라우드 운영(AWS, CI/CD, GitOps, Infrastructure as Code) 등을 학습하고, 실무 환경에서 사용되는 최신 기술을 활용하여 프로젝트를 수행한다.
